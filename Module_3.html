1. What Are K-maps?

A Karnaugh Map (K-Map) is a graphical method used to simplify Boolean expressions and reduce the number of logic gates required in a circuit.

Steps to solve K-maps (SOP Minimisation) 

1.Determine the number of variables:
        If it is given in question as , F(A,B) it means that there are 2 variables
        If it is given in question as, F(A,B,C) it means that there are 3 variables and so on
For F(A,B) → 2 variables → 2×2 K-map (4 cells)
For F(A,B,C) → 3 variables → 2x4 K-map (8 cells)
For F(A,B,C,D) → 4 variables → 4x4 K-map (16 cells)
For F(A,B,C,D,E) → 5 variables → 2(4x4) K-map (32 cells)

Note : Since construction of 32 cells isn't possible to draw on paper we divide the 5 Variable K-map into two equivalent four variable maps with A,B,C,D and E
Variable A distinguishes between the two maps  A= 0; A=1
2.Draw appropriate K-Map(Using Gray Code order)
3.Fill the K-Map
4.Form Groups(Group sizes are in powers of 2)
5.Always prefer : 8-cell groups > 4-cell groups > 2-cell groups > 1-cell groups
Example:
Instead of two pairs, make one quad if possible.
The larger the group, the simpler the answer.
6.Use Wrap-Around Adjacency:
Left edge touches right edge.
Top edge touches bottom edge.
Corners are adjacent.
Example:
1 0 0 1
The first and last pairs cells can form a pair.
If overlapping is possible then allow it.
7.Find the common variables in each group(If value of the variables are changing then remove them)
Example:
In two groups we have as, 
A = 0, B = 0, C = 1
A = 0, B = 1, C = 1
Here, B is changing of its values, So remove B, Finally we get as
A'C
8.SOP all the answers

2. Two Variable K-Maps

It has 2 variables consisting of 4 cells (2×2 Cells)

Remember this table

A\B 0 1
0 m₀ m₁
1 m₂ m₃

The MinTerm values are :
m₀ = A'B'
m₁ = A'B
m₂ = AB'
m₃ = AB

6. K-Maps with donot care(SOP)

A don't care condition is an input combination for which the output is not important or never occurs in practice.

It is represented by X or d.

Steps to Remember

1. Draw the K-Map
2. Fill the K-Map
Put 1 in minterms
Put X or d in donot care condition
Put 0 elsewhere
Ex: F(A,B,C)=Σm(1,3,7) donot care d(5)
A\BC 00 01 11 10
0 0 1 1 0
1 0 X 1 0 
3. Form groups (Try to include X if it forms a larger group)
Without using X
1 1 
X 1 


With using X
1 1 
1 1

We have to treat X as 1 for getting a larger group 
4. Find common variables

Note : This is for SOP, If you want to calculate for POS replace 1 with 0 , group 0's Use X as 0 if needed, otherwise ignore X

7. Problems on K-maps

2-Variable K-Map Problems
1. F(A,B)=Σm(1,3)

A\B 0 1
0 0 1
1 0 1

Group m₁ and m₃
m₁ = A'B
m₃ = AB
Common terms in both are B
Hence the final answer is (SOP)
F = B

-------------------------------------------------------------------------------------------------------------------------------------------------------------

2. F(A,B)=Σm(2,3)

A\B 0 1
0 0 0
1 1 1

Group m₂ and m₃
m₂ = AB'
m₃ = AB
Common terms in both are A
Hence the final answer is(SOP)
F = A

-------------------------------------------------------------------------------------------------------------------------------------------------------------

3. F(A,B)=Σm(1,2,3)

A\B 0 1
0 0 1
1 1 1

Group m₁ and m₃, also m₂ m₃
m₁ = A'B
m₃ = AB
The common terms in both are B
m₂ = AB'
m₃ = AB
The common terms in both are A
Hence the final answer(SOP) is A + B

3 Variable K-Maps

1. F(A,B,C)=Σm(1,3,5,7)

A\BC 00 01 11 10
0 0 1 1 0
1 0 1 1 0

Here m₁, m₃, m₅, m₇ form a quad (Instead of taking two at once the preference is always check for 8 > 4 > 2 > 1)
Hence the common terms in them are C
Hence the final answer(SOP) is C

-------------------------------------------------------------------------------------------------------------------------------------------------------------

2. F(A,B,C)=Σm(2,3,6,7)

A\BC 00 01 11 10
0 0 0 1 1
1 0 0 1 1

Here again it forms a quad
m₃, m₂, m₇, m₆
m₃ = A'BC
m₂ = A'BC'
m₇ = ABC
m₆ = ABC'
The common terms are B
Hence the final answer(SOP) is B
