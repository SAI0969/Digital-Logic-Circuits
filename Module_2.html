1. What Are Booleans for Logic Gates?

A Boolean variable has only two possible values: 0 or 1. In digital logic, these values represent two voltage states.

Boolean 0

False / LOW / OFF

Means the condition is not true. In a circuit, it commonly represents a low voltage level.

Boolean 1

True / HIGH / ON

Means the condition is true. In a circuit, it commonly represents a high voltage level.

Logic Gate Idea

A logic gate accepts Boolean inputs and produces one Boolean output. Example: an AND gate outputs 1 only when all inputs are 1.

Symbol Meaning Common name
A + B A OR B Logical addition
A · B or AB A AND B Logical multiplication
A' or ¬A NOT A Complement

2. Axiomatic Definition of Boolean Algebra

Boolean algebra is an algebraic system over a set B with two binary operations OR (+), AND (·), one unary operation NOT ('), and two special elements 0 and 1.

Identity Elements

A + 0 = A
A · 1 = A

Complement Laws

A + A' = 1
A · A' = 0

Idempotent Laws

A + A = A
A · A = A

Distinct Elements

0 ≠ 1

The two Boolean values must be different.

Commutative Laws

A + B = B + A
A · B = B · A

Associative Laws

A + (B + C) =
(A + B) + C
A(B·C) = (AB)·C

Distributive Laws

A · (B + C) =
A·B + A·C
A + (B · C) =
(A+B)·(A+C)

DeMorgan's Laws

(A + B)' = A' · B'
(AB)' = A' + B'

Absorption Laws

A + A·B = A
A(A + B) = A

Domination Laws (or) Null Laws

A + 1 = A
A · 0 = 0

Involution Law (or) Double Complement Law

(A')' = A

4. Boolean Functions

A Boolean function is an expression made from Boolean variables, constants 0 and 1, and operations AND, OR, and NOT. The output is always 0 or 1.

Example Function

F(A,B,C) = A'B + AC

The function has three input variables. For every input combination, F has one output.

Truth Table

A truth table lists every input combination and the output of the function.

For 3 variables there are 2^3 = 8 rows.

Literal

A literal is a variable or its complement, such as A or A'.

5. Minterms and Maxterms

Minterm

A minterm is an AND term containing every variable exactly once, either complemented or uncomplemented. It is 1 for exactly one truth-table row.

For A,B,C and row 101:
A=1, B=0, C=1
Minterm = A·B'·C = m5

Maxterm

A maxterm is an OR term containing every variable exactly once. It is 0 for exactly one truth-table row.

For A,B,C and row 101:
A=1, B=0, C=1
Maxterm = A' + B + C' = M5
Index A B C Minterm(Result = 1) Maxterm(Result = 0)
0 0 0 0 A'B'C' A+B+C
1 0 0 1 A'B'C A+B+C'
2 0 1 0 A'BC' A+B'+C
3 0 1 1 A'BC A+B'+C'
4 1 0 0 AB'C' A'+B+C
5 1 0 1 AB'C A'+B+C'
6 1 1 0 ABC' A'+B'+C
7 1 1 1 ABC A'+B'+C'

Generalisation of Min terms easily

To form a minterm, take every variable. If a variable's value in the given row is 0, complement it; if it is 1, leave it unchanged. Then AND all variables together. The resulting expression evaluates to 1 only for that row.
Example :
Let us analyse for Index 5 

A = 1; B = 0; C = 1.

As mentioned above if a variable value is 0 take its compliment, we have B value as 0.

Hence take its compliment -> B' the other variable values are 1 hence leave it unchanged.

Now, we need to AND all these variables AB'C.

It is denoted as m5.



Verification:

Now this result becomes A = 1, B' = 1, C = 1 AND gate means doing multiplication.

All the variable values must be 1, then only the result is 1

Generalisation of Max terms easily

To form a maxterm, take every variable. If a variable's value in the given row is 1, complement it; if it is 0, leave it unchanged. Then OR all variables together. The resulting expression evaluates to 0 only for that row.
Example :
Let us analuse for Index 5 

A = 1; B = 0; C = 1.

As mentioned above if a variable value is 1 take its compliment, we have both A, C as 1.

Hence take its compliment -> A', C' the other variable B value is 1 hence leave it unchanged.

Now, we need to OR all these variables A'+B+C'.

It is denoted as M5.



Verification:

Now this result becomes A' = 0, B = 0, C' = 0 OR gate means doing addition.

All the variable values must be 0, then only the result is 0

Easy Rule: Find either minterm or maxterm, the other will be the compliment of other
 Ex: If we know the minterm of m5 as AB'C, for the maxterm of M5 do compliment of m5
 (AB'C)' → A' + B + C'

6. Canonical and Standard Forms

Canonical SOP

Sum of minterms. Use truth-table rows where F=1.

F(A,B,C) = Σm(1,3,5)
= A'B'C + A'BC + AB'C

Canonical POS

Product of maxterms. Use truth-table rows where F=0.

F(A,B,C) = ΠM(0,2,4,6,7)

Standard SOP

OR of product terms. Terms do not have to contain every variable.

F = A'B + AC

Standard POS

AND of sum terms. Terms do not have to contain every variable.

F = (A+B')(A'+C)
Canonical means complete: every term contains all variables. Standard means structured: SOP is OR of AND terms; POS is AND of OR terms.

7. SOP to POS and POS to SOP Conversion

Canonical SOP and canonical POS describe the same Boolean function in opposite ways. SOP lists the truth-table rows where F=1. POS lists the truth-table rows where F=0.

SOP to POS

  1. Find the total number of rows: 2n, where n is the number of variables.
  2. Write all row indexes from 0 to 2n-1.
  3. Remove the SOP minterm indexes.
  4. The remaining indexes become the POS maxterm indexes.
F(A,B,C) = Σm(1,3,5)
All indexes: 0,1,2,3,4,5,6,7
Missing indexes: 0,2,4,6,7

So F(A,B,C) = ΠM(0,2,4,6,7)

POS to SOP

  1. Find the total number of rows.
  2. Write all row indexes.
  3. Remove the POS maxterm indexes.
  4. The remaining indexes become the SOP minterm indexes.
F(A,B,C) = ΠM(0,2,4,6,7)
All indexes: 0,1,2,3,4,5,6,7
Missing indexes: 1,3,5

So F(A,B,C) = Σm(1,3,5)
Quick rule: For the same function, minterm indexes and maxterm indexes are complements of each other over the full truth table.

8. Digital Logic Gates

Digital gates implement Boolean operations in hardware.

Gate Expression Output is 1 when,
NOT Y = A' A is 0
AND Y = A·B A and B are both 1
OR Y = A+B At least one input is 1
NAND Y = (A·B)' Not both inputs are 1
NOR Y = (A+B)' Both inputs are 0
XOR Y = A⊕B Inputs are different
XNOR Y = (A⊕B)' Inputs are the same

9. Synthesis Using AND, OR, and NOT Gates

Synthesis means building a circuit from a Boolean function or truth table.Most commonly we use SOP form for easy and quicker building of the gates.(As, We know that SOP uses minterm, Hence it is easier to build gates.)

  1. Make the truth table.
  2. Find all rows where F=1.(Most of the time where F becomes 1 is given in question, If in question you are asked to make for F becomes 0 then use maxterm)
  3. Write one minterm for each F=1 row.
  4. OR the minterms together to get SOP.
  5. Build each minterm with NOT and AND gates, then combine them with OR gates.
Question1: Realize the Boolean function F(A,B,C)=Σm(1,3,5) using AND, OR, and NOT gates.

In above question1 it is given that, F(A,B,C)=Σm(1,3,5) where Σm means that for those indices of F the values are 1.Since here Σm is used which means that it is minterm hence do OR gate

10. NAND and NOR Logic Networks

NAND and NOR are called universal gates because either one can be used to build NOT, AND, and OR gates.

Using NAND Only

NOT A = A NAND A
A AND B = (A NAND B) NAND (A NAND B)
A OR B = (A NAND A) NAND (B NAND B)

Using NOR Only

NOT A = A NOR A
A OR B = (A NOR B) NOR (A NOR B)
A AND B = (A NOR A) NOR (B NOR B)
NAND-NAND networks naturally implement SOP forms. NOR-NOR networks naturally implement POS forms.
