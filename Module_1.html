1. Number System Basics and Conversions

Decimal

Base 10

Digits 0 to 9. Place values are powers of 10. 

548 = 5 x 10^2 + 4 x 10^1 + 8 x 10^0

Binary

Base 2

Digits 0 and 1. Place values are powers of 2. 

1011₂ = 1x8 + 0x4 + 1x2 + 1x1 = 11₁₀

Octal

Base 8

Digits 0 to 7. One octal digit equals 3 binary bits. 

725₈ = 7x64 + 2x8 + 5 = 469₁₀

Hexadecimal

Base 16

Digits 0 to 9 and A to F. One hex digit equals 4 binary bits. 

2A₁₆ = 2x16 + 10 = 42₁₀

Decimal to Any Base

  1. Divide the decimal number by the target base.
  2. Write the remainder.
  3. Repeat with the quotient until it becomes 0.
  4. Read remainders from bottom to top. 
45₁₀ to binary:
45/2=22 r1
22/2=11 r0
11/2=5  r1
5/2=2   r1
2/2=1   r0
1/2=0   r1
Answer: 101101₂

Any Base to Decimal

Multiply each digit by its positional weight and add. 

11001₂ = 1x2^4 + 1x2^3 + 0x2^2 + 0x2^1 + 1x2^0
= 16 + 8 + 0 + 0 + 1 = 25₁₀

Binary, Octal, Hex Shortcuts

Binary to Octal

Group bits in 3s from the right. 

1011011₂ = 001 011 011 = 133₈

Octal to Binary

Replace each octal digit with 3 bits. 

57₈ = 5 7 = 101 111 = 101111₂

Binary to Hex

Group bits in 4s from the right. 

11010110₂ = 1101 0110 = D6₁₆

Hex to Binary

Replace each hex digit with 4 bits. 

3A₁₆ = 0011 1010₂

2. BCD Conversion

BCD means Binary Coded Decimal. Each decimal digit is converted separately into 4-bit binary.

Decimal digit BCD
0 0000
1 0001
2 0010
3 0011
4 0100
5 0101
6 0110
7 0111
8 1000
9 1001

Decimal to BCD 

529₁₀:
5 = 0101
2 = 0010
9 = 1001
Answer: 0101 0010 1001

BCD to Decimal 

0100 0111 1000:
0100=4, 0111=7, 1000=8
Answer: 478₁₀
BCD groups 1010 to 1111 are invalid because decimal digits only go from 0 to 9.

3. BCD Addition

Add normally in binary, but every 4-bit BCD digit must be 0000 to 1001.

  1. Add the BCD digits using binary addition.
  2. If the result is greater than 1001, add 0110.
  3. If there is a carry out, add 0110 and pass the carry to the next digit. 
8 + 7:
  1000
+ 0111
------
  1111  invalid BCD
+ 0110  correction
------
1 0101
Answer: 0001 0101 = 15

Example: 29 + 38 

29 = 0010 1001
38 = 0011 1000
Ones: 1001 + 1000 = 1 0001, correction gives 0111 and carry 1
Tens: 0010 + 0011 + 1 = 0110
Answer: 0110 0111 = 67

5. Binary to Gray and Gray to Binary

Binary to Gray

  1. Copy the first binary bit.
  2. XOR each binary bit with the previous binary bit. 
Binary 1011:
first Gray bit = 1
1 XOR 0 = 1
0 XOR 1 = 1
1 XOR 1 = 0
Gray = 1110

Gray to Binary

  1. Copy the first Gray bit.
  2. XOR the previous binary result bit with the current Gray bit. 
Gray 1110:
first binary bit = 1
1 XOR 1 = 0
0 XOR 1 = 1
1 XOR 0 = 1
Binary = 1011
XOR rule: same bits give 0; different bits give 1.

6. Complements

1's Complement

Change every 0 to 1 and every 1 to 0. 

Binary:          101100
1's complement: 010011

2's Complement

Find 1's complement, then add 1. 

Binary:          101100
1's complement: 010011
Add 1:           000001
2's complement: 010100

2's Complement Shortcut

From the right, copy bits up to and including the first 1, then complement all remaining bits to the left. 

101100 -> copy __1100, complement 10 to 01
Answer: 010100

Binary Subtraction Using 2's Complement

  1. Make both numbers the same bit length.
  2. Find the 2's complement of the number being subtracted.
  3. Add it to the first number.
  4. If carry occurs, discard it. The answer is positive.
  5. If no carry occurs, take 2's complement of the result and write a minus sign. 
13 - 5:
13 = 1101
5  = 0101
2's complement of 0101 = 1011
  1101
+ 1011
------
1 1000
Discard carry. Answer = 1000₂ = 8₁₀

Negative Example: 5 - 13 

5 = 0101, 13 = 1101
2's complement of 1101 = 0011
0101 + 0011 = 1000
No carry, so result is negative.
2's complement of 1000 = 1000
Answer = -1000₂ = -8₁₀

Binary Multiplication

Use long multiplication. Multiplying by 0 gives 0; multiplying by 1 copies the number. 

1011 x 101:
      1011
   x   101
   -------
      1011
     0000
    1011
   -------
    110111

Binary Division

Use long division: compare, subtract, bring down the next bit. 

1101₂ / 11₂:
11 goes into 110 one time: 110 - 011 = 011
Bring down 1: 111
11 goes into 111 one time: 111 - 011 = 100
Quotient = 11₂, Remainder = 100₂
Check: 13 / 3 = 3 remainder 4
